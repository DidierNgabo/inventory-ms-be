## Description

this is the backend application of household equipment repair management system  

## Installation

```bash
$ clone this repository
$ create a new postgres database with the name you want
$ npm install
$ create a .env file using  the .env.example file in the root of the project
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
